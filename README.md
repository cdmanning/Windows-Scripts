## üìú Available Scripts

| Script Name | Purpose |
| :--- | :--- |
| **Winget-Upgrade.bat** | Installs all available updates for installed applications that are tracked by the Windows Package Manager. |
| **Repair-System-Files.bat** | Checks for corrupted system files after checking for administrator privileges. |
| **Launch-SystemConfiguration.bat** | Opens the Windows "System Configuration" window to change system boot settings. |
| **Open-Environment-Variables.bat** | Opens the Windows Environment Variables editor with administrator privileges. |
| **Network-Test.bat** | Pings Google's public DNS to check for internet connection. |
| **Rebuild-IconCache.bat** | Rebuilds the icon-cache and refreshes the Explorer service. |
| **Launch-Memory-Diagnostics.bat** | Runs the built-in Memory Diagnostics tool checking for issues with the installed system memory after a restart. |
| **Restart-Computer.bat** | Forces a system restart of the computer. |
| **Shutdown-Computer.bat** | Forces a system shutdown of the computer. |
| **Quick-Access-Listening-Settings.bat** | Open settings to input sound devices to turn on/off listening. |
| **Open-Installed-Apps-Folder.bat** | Opens the Windows directory where the shortcuts to installed applications can be found. |
| **Identify-P2P-Connections.bat** | Used to identify Peer-To-Peer connections by matching an executable's PID in the Windows Task Manager with the listed connections. |
| **Create-Invisible-Folder.bat** | Creates a **invisible folder** in the current directory without an icon or name. |
